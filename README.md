# Atividade Avaliativa: Lógica de Programação com Python 🐍

Este repositório contém a resolução de uma série de exercícios práticos focados em lógica de programação, estruturas de repetição (loops) e condicionais, desenvolvidos em ambiente Jupyter Notebook (Google Colab).

## 📋 Descrição do Projeto

O objetivo deste projeto é aplicar conceitos fundamentais de Python para resolver problemas do cotidiano, como análise de dados climáticos, gestão escolar, monitoramento de energia e simulação de e-commerce.

## 🚀 Desafios Resolvidos

### 1. Classificação de Temperaturas
Análise de uma lista de temperaturas semanais para classificação em categorias (Frio, Agradável e Quente) e contagem estatística de ocorrências.

### 2. Sistema de Avaliação de Alunos
Processamento de notas escolares para determinar o status acadêmico (Aprovado, Reprovado ou Recuperação), incluindo o cálculo do percentual de reprovação da turma.

### 3. Monitoramento de Consumo de Energia
Filtragem de dados de consumo (kWh) para identificação de picos de uso, cálculo de média semanal e sistema de alerta para excesso de consumo alto.

### 4. Simulação de Carrinho de Compras
Aplicação de regras dinâmicas de desconto baseadas no valor de cada item, cálculo do valor final da compra e mensuração da economia total obtida.

## 🛠️ Tecnologias Utilizadas

* **Python 3**
* **Google Colab / Jupyter Notebook**
* **Manipulação de Listas e Estruturas de Controle**
